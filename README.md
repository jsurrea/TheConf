# TheConf

Welcome! This repo is part of my learning adventure on frontend technologies. It was developed mobile-first using Bootstrap 5.3 and includes several key features of the framework such as:

- Scrollspy
- Carousel
- Modals
- Tooltips
- Alerts
- Cards
- Navbars
- Forms
- and more!
